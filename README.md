# Lista de Tareas - Versión Node.js

Esta es una versión de consola de la lista de tareas, convertida para ejecutarse con Node.js.

## Instalación

1. Si no tienes Node.js instalado, descárgalo desde [nodejs.org](https://nodejs.org/) e instálalo.

2. Verifica la instalación ejecutando `node --version` en la terminal.

## Uso

1. Abre una terminal en la carpeta del proyecto (`c:\Users\cir\Desktop\to-do list`).

2. Ejecuta el comando: `node app.js`

3. Sigue las instrucciones en pantalla para agregar, listar o eliminar tareas.

## Funcionalidades

- **Agregar tarea**: Ingresa una nueva tarea.
- **Listar tareas**: Muestra todas las tareas numeradas.
- **Eliminar tarea**: Elimina una tarea por su número.
- **Salir**: Cierra la aplicación.

Las tareas se almacenan en memoria durante la sesión. Al cerrar, se pierden (puedes modificar el código para guardar en archivo si quieres).
